**formatEPCPayload({ iban, name, amount, bic, remittance })** — returns string (QRCD-01):
- Builds exactly 12 fields joined by `'\n'` (LF only, never CRLF):
  - Line 1: `'BCD'` (service tag, hardcoded)
  - Line 2: `'002'` (version 2, BIC optional)
  - Line 3: `'1'` (character set = UTF-8)
  - Line 4: `'SCT'` (SEPA Credit Transfer)
  - Line 5: `(bic || '').trim().substring(0, 11)` (BIC, blank if omitted)
  - Line 6: `name.substring(0, 70)` (beneficiary name, max 70 chars)
  - Line 7: `iban.replace(/\s/g, '').toUpperCase()` (IBAN, normalized)
  - Line 8: `amount != null ? 'EUR' + amount.toFixed(2) : ''` (amount or empty if unknown)
  - Line 9: `''` (purpose code, always blank)
  - Line 10: `''` (structured remittance, always blank)
  - Line 11: `(remittance || '').substring(0, 140)` (unstructured remittance)
  - Line 12: `''` (originator info, always blank)
- Return `fields.join('\n')`
- Guard: if `iban` is falsy or `name` is falsy, throw an Error with message `'IBAN and name are required'`

**validateEPCPayload(payload)** — returns object (QRCD-03):
- Count bytes: `new TextEncoder().encode(payload).length`
- Return `{ valid: byteCount <= 331, byteCount: byteCount, errors: [] }` where errors array contains `'Payload exceeds 331-byte EPC limit (N bytes)'` if over limit
- Also check payload starts with `'BCD\n002\n1\nSCT\n'` — if not, add error `'Invalid EPC header'`

**parsePaymentText(text)** — returns object:
- Orchestrator that calls all extractors in sequence:
  1. `const normalized = normalizeText(text)`
  2. `const iban = extractIBAN(normalized)`
  3. `const amount = extractAmount(normalized)`
  4. Find IBAN position for name proximity: `const ibanIndex = iban ? normalized.toUpperCase().indexOf(iban.substring(0, 4)) : -1`
  5. `const name = extractName(normalized, ibanIndex)`
  6. Return `{ iban: iban, name: name, amount: amount }`

All three functions must be at top-level scope (no import/export).
No fetch/XHR/network calls (GENL-03).

**Part C: Add renderQR function**

Add to the main `<script>` block, after validateEPCPayload:

```javascript
function renderQR(payloadString, containerElement) {
  if (!containerElement) {
    containerElement = document.getElementById('qr-output');
  }
  var qr = qrcode(0, 'M');  // typeNumber 0 = auto-detect; ECC level M
  qr.addData(payloadString);
  qr.make();
  containerElement.innerHTML = qr.createSvgTag({ cellSize: 2, margin: 4, scalable: true });
  return containerElement.innerHTML;
}
```
